## Self-Critique Framework — How to Improve Painting

### The lesson (learned from user feedback)

The user repeatedly pushed back:
- "Tu reproduis mais tu n'apportes rien de plus" → stop copying, start interpreting
- "C'est beaucoup trop simple, on ne reconnait rien" → 38 strokes is too few
- "Tes reproductions sont des copies floues" → need detail, not just color averages

### The cycle that works

1. **Paint** something
2. **Critique** it honestly — what's wrong specifically?
3. **Diagnose** the root cause (not the symptom)
4. **Fix** with targeted changes
5. **Repeat**

### Critique categories

Ask these questions after every painting:

**Composition**: Is the subject recognizable? Is there visual hierarchy (focal point vs background)?
**Value**: Are shadows dark enough? Highlights bright enough? Is there enough contrast?
**Color**: Are colors accurate? Too saturated? Too desaturated? Warm/cool balance?
**Edges**: Are boundaries defined? Soft where they should be? Sharp where they should be?
**Texture**: Is it too smooth (blurry)? Too rough (noisy)? Just right (painterly)?
**Detail**: Is there enough detail to recognize the subject? Too much (pixelated)?

### The stroke count sweet spot

Too few strokes (< 100): Unrecognizable abstraction
Too many strokes (> 15,000): Blurry pixelated copy
Sweet spot (500-5,000): Recognizable painting with visible brushwork

For the best balance:
- Base layer: 200-1000 strokes (broad color zones)
- Detail layer: 1000-3000 strokes (refinement)
- Edge layer: 500-2000 strokes (structural detail)
- Style layer: 50-200 strokes (accents, texture)
- Total: 2000-6000 strokes

### What to do when stuck

If SSIM plateaus:
1. Check if it's a VALUE problem (too little contrast)
2. Check if it's an EDGE problem (missing boundaries)
3. Check if it's a COLOR problem (wrong hues)
4. Check if it's a DETAIL problem (blocks too large)

Each problem has a different fix. Don't just add more of the same strokes.
